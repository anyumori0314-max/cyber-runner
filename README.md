# 避けろ！サイバーランナー

PCブラウザで遊べるシンプルな避けゲームです。

## ゲーム概要

プレイヤーは画面下部のキャラクターを左右に動かし、上から落ちてくる障害物を避け続けるゲームです。

### 特徴
- **難易度の段階的上昇**: 時間が経つほど障害物の落下速度が速くなり、出現間隔が短くなります
- **スコアシステム**: 生存時間に応じてスコアが加算されます
- **レベル表示**: ゲーム進行度を示すレベルが画面に表示されます
- **ハイスコア保存**: ハイスコアはブラウザのローカルストレージに自動保存されます
- **サイバー風デザイン**: ネオン色とグロー効果でサイバーな雰囲気を演出しています

## 操作方法

### キーボード操作
- **左移動**: `←（左矢印キー）` または `A キー`
- **右移動**: `→（右矢印キー）` または `D キー`

### ボタン操作
- **START**: タイトル画面からゲーム開始
- **RETRY**: ゲームオーバー後に再プレイ

## 起動方法

### 方法1: ファイルをブラウザで直接開く
1. `index.html` をダブルクリック、またはブラウザにドラッグ&ドロップ
2. ゲーム画面が表示されます

### 方法2: ローカルサーバーで実行（推奨）
```bash
# Python 3の場合
python -m http.server 8000

# Python 2の場合
python -m SimpleHTTPServer 8000

# Node.jsの場合
npx http-server
```

その後、ブラウザで `http://localhost:8000` を開いてください。

cyber-runner/

### ファイルの説明

#### index.html
- ゲームの画面構成を定義
- 3つの画面を含む:
  - タイトル画面
  - ゲーム画面
  - ゲームオーバー画面

#### style.css
- サイバー風のネオンデザインを実装
- グラデーション背景
- グロー効果やアニメーション
- レスポンシブデザイン対応

#### script.js
- ゲームロジックの全実装
- プレイヤーの操作
- 障害物の生成と衝突判定
- スコア計算と難易度管理
- localStorageによるハイスコア保存

## ゲームの進め方

1. **タイトル画面**: ゲーム説明とハイスコアが表示されます
2. **START ボタンをクリック**: ゲーム開始
3. **プレイ中**: 
   - 左右キーで操作
   - 落ちてくる障害物を避ける
   - スコアとレベルが画面に表示される
4. **ゲームオーバー**: プレイヤーが障害物に衝突すると終了
5. **最終スコア表示**: 最終スコアとハイスコアが表示
6. **RETRY ボタン**: ゲーム再プレイ

## 難易度の上がり方

### 1. 落下速度
- 初期速度: 3
- 最高速度: 12
- 60秒でプログレッシブに加速

### 2. 出現間隔
- 初期出現率: 2%
- 最大出現率: 8%
- 60秒でプログレッシブに増加

### 3. レベル
- 生存時間に基づいて計算
- 10秒ごとにレベルが1上がる

## 技術仕様

### 使用技術
- **HTML5**: Canvas API（ゲーム描画）
- **CSS3**: グラデーション、アニメーション、グロー効果
- **JavaScript（Vanilla）**: 外部ライブラリなし

### ゲーム定数（script.js内）
```javascript
CANVAS_WIDTH = 800          // キャンバス幅
CANVAS_HEIGHT = 600         // キャンバス高さ
PLAYER_WIDTH = 40           // プレイヤーサイズ
OBSTACLE_WIDTH = 50         // 障害物サイズ
INITIAL_SPEED = 3           // 初期落下速度
MAX_SPEED = 12              // 最大落下速度
INITIAL_SPAWN_RATE = 0.02   // 初期出現率（2%）
MAX_SPAWN_RATE = 0.08       // 最大出現率（8%）
```

## データ保存

このゲームはブラウザの **localStorage** を使用してハイスコアを保存しています。

- **保存先**: ブラウザのローカルストレージ
- **キー**: `cyberRunnerHighScore`
- **自動保存**: ゲームオーバー時に新しいハイスコアがあれば自動保存

### ハイスコアをリセットする方法
ブラウザのコンソール（F12キー）で以下を実行：
```javascript
localStorage.removeItem('cyberRunnerHighScore');
```

## トラブルシューティング

### ゲームが起動しない
1. `index.html` をテキストエディタで開き、ファイルが正しいか確認
2. ブラウザのコンソール（F12）でエラーメッセージを確認
3. ローカルサーバーで実行してみる

### キーが反応しない
- ブラウザウィンドウにフォーカスがあることを確認
- 矢印キーまたは A/D キーが対応しているか確認

### ハイスコアが反映されない
1. ブラウザが localhost サーバーで実行しているか確認
2. ブラウザのシークレットモードで実行していないか確認
3. コンソールで以下を実行: `console.log(localStorage.getItem('cyberRunnerHighScore'))`
